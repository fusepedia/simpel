## 📋 Deskripsi

**SIMPEL** adalah sistem informasi berbasis web untuk mengelola peminjaman
peralatan dan penggunaan bahan pelatihan di Jurusan Elektronika BPVP Sorong.
Sistem ini mendukung dua program pelatihan:

- 📱 **Teknisi Telepon Seluler**
- 🔌 **Instalasi Jaringan Fiber Optik**

---

## ✨ Fitur Utama

| Fitur | Deskripsi |
|-------|-----------|
| 📊 Dashboard | Ringkasan statistik real-time |
| 🔧 Inventaris Peralatan | Kelola 20 jenis peralatan |
| 📦 Bahan Pelatihan | Tracking 20 jenis bahan habis pakai |
| 📝 Form Peminjaman | Multi-step form dengan validasi |
| 👁️ Preview Dokumen | Preview dokumen sebelum cetak |
| 📄 Export PDF | Download formulir peminjaman sebagai PDF |
| 🖨️ Print | Cetak langsung dari browser |
| 📚 Kurikulum | Kurikulum lengkap sesuai program Kemnaker |
| 🔍 Pencarian | Filter dan cari data secara real-time |
| 📱 Responsive | Tampilan optimal di semua perangkat |
| 🌙 Dark Mode | Tema gelap modern |

## 🚀 Cara Menggunakan

### Deploy via GitHub Pages

1. Fork repository ini
2. Buka **Settings** → **Pages**
3. Pilih Branch: `main`, Folder: `/ (root)`
4. Klik **Save**
5. Akses di: `https://username.github.io/SIMPEL-BPVP-Sorong/`
